# Kernel driver ir35221

Supported chips:
:   * Infineon IR35221

      Prefix: ‘ir35221’

      Addresses scanned: -

      Datasheet: Datasheet is not publicly available.

Author: Samuel Mendoza-Jonas <[sam@mendozajonas.com](mailto:sam%40mendozajonas.com)>

## Description

IR35221 is a Digital DC-DC Multiphase Converter

## Usage Notes

This driver does not probe for PMBus devices. You will have to instantiate
devices explicitly.

Example: the following commands will load the driver for an IR35221
at address 0x70 on I2C bus #4:

```
# modprobe ir35221
# echo ir35221 0x70 > /sys/bus/i2c/devices/i2c-4/new_device
```
