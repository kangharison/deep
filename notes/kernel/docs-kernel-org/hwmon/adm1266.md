# Kernel driver adm1266

Supported chips:
:   * Analog Devices ADM1266
      Prefix: ‘adm1266’
      Datasheet: <https://www.analog.com/media/en/technical-documentation/data-sheets/ADM1266.pdf>

Author: Alexandru Tachici <[alexandru.tachici@analog.com](mailto:alexandru.tachici%40analog.com)>

## Description

This driver supports hardware monitoring for Analog Devices ADM1266 sequencer.

ADM1266 is a sequencer that features voltage readback from 17 channels via an
integrated 12 bit SAR ADC, accessed using a PMBus interface.

The driver is a client driver to the core PMBus driver. Please see
[Kernel driver pmbus](pmbus.html) for details on PMBus client drivers.

## Sysfs entries

The following attributes are supported. Limits are read-write, history reset
attributes are write-only, all other attributes are read-only.

inX\_label “voutx”
inX\_input Measured voltage.
inX\_min Minimum Voltage.
inX\_max Maximum voltage.
inX\_min\_alarm Voltage low alarm.
inX\_max\_alarm Voltage high alarm.
